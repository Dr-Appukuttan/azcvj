Parrot Security OS

to its looks.It has a simple UI. Also it has a Support Group where u can ask the errors you face and thus things made much more easy.  Only the thing is it do not support ARM devices thus it is not availabel to Pi users.  If you got a 4 gig RAM and wanna do pentesting then go for Parrot Security OS.  Its home version is also available.  Refer https://www.parrotsec.org for more details.



Arch Linux
It is a highly lightweight Distro. It can run on even a 1 gig RAM system
Only the hard part is that it's installation and setup is a bit hard and you need to be a master linux inorder to use it. But with the help of Google or DuckDuckGO you can set it up. And it has AUR repo from where you can install all tools.  If you customize it and install some of those hacking tools u want then , it will be a perfect distro for hacking.  Like for Parrot , it also has a Support Group where you can ask your errors and solve 'em.  As i said the only problem is it is not having a GUI installer(but can be installed from AUR.I haven't tried) and you gotta do all manually.
Visit https://www.archlinux.org for more details
